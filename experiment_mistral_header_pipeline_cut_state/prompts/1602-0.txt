
1. **Informal Theorem**  
Element at Index in Range List Equals Index : For any natural number \( n \) and any natural number \( m \) such that \( m < n \), the element at index \( m \) in the list `List.range n` is equal to \( m \). In other words, \((\text{List.range } n)[m] = m\).

2. **Informal Proof**  
We start by noting that the list generated by `List.range n` is equal to the list generated by `List.range' 0 n`. Both lists contain the natural numbers from \( 0 \) to \( n-1 \) in increasing order. Therefore, the element at index \( m \) in `List.range n` is the same as the element at index \( m \) in `List.range' 0 n`, which is \( m \). This simplification directly shows that \((\text{List.range } n)[m] = m\). Hence, the theorem is proved.
